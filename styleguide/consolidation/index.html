About headings:

There are 4 lairs of headings are used: h1, h2, h3, h4. H1 is exclusively used in the banner and can be a link to the home menu on smaller devices, H2 generally highlights the major sections of the site and major subsections. H3 is used to highlight devisions in the subs sections such as what's a form what's a picture and so on. H4 is for other miscellaneous sections that would require bitter titles. for sections smaller than h4, use the em or strong tag to bold elements

Sample code for h1:

<div class=banner><h1>Orbitar</h1></div>

Orbitar

Main Navigation

N/A

The main navigation for this website is always in the banner and can be accessed easily from there (see navigation button). There aren't other navigation buttons on the site as the site will react and change in shape in accordance to user clicks. There will be windows on the site which will expand and collaps when another window is clicked. this means that the user has access to all other parts of the site at any time
