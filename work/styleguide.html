Headings

CSS selectors

h2
h3

About headings:

There are 4 lairs of headings are used: h1, h2, h3, h4. H1 will lable different modules of the main site, H2 generally highlights the major sections of the site and major subsections of modules. H3 is used to highlight divisions in the subs sections such as what's a form what's a picture and so on. H4 is for other miscellaneous sections that would require bitter titles. For sections smaller than h4, use the em or strong tag to bold elements

Sample code for H1:


            <div class="banner">
            <h1>Orbitar</h1>
            </div>

Rendered Element:

Orbitar

Main Header Navigation

CSS selectors

#navHeader

The header navigation bar is the primary method of accessing parts of the website. As our website is primarily a one-page site, many of the links will be anchor links to the rest of the same page if the user is accessing from the primary main page.

The navigation bar on the actual site has its position set as fixed, so it will always be at the top of the browser window / screen. For readability, the example below has its CSS overridden to position: static. In addition, the grid breakpoints that enable this nav bar to be responsive on the actual site causes it to behave slightly differently on this style guide's stylesheet.

Sample Code:


<nav id="navHeader"> <!-- banner -->
    <div id="centeredNavContainer">
        <div id="navBar">
            <div id="navBarLeft">
                <span id="siteName"><a href="mainpage.html">Orbitar</a></span>
            </div> <!-- ./navBarLeft -->
            <div id="navBarRight">
                <ul class="dropDownContainer">
                    <!-- when any of these buttons are clicked, close the menu -->
                    <li><a class="buttonNav dropDownButton" href="#" >About</a></li>
                    <li><a class="buttonNav dropDownButton" href="#" >Create / Modify Booking</a></li>
                    <li><a class="buttonNav dropDownButton" href="#" >Feedback</a></li>
                </ul>
            </div> <!-- ./navBarRight -->
        </div> <!-- ./navBar -->
    </div> <!-- ./centered-nav-container -->
</nav>

Rendered Element

Orbitar
  • About
  • Create / Modify Booking
  • Feedback

Product Listing

Orbitar provides space transportation, accommodation, and research services, so this is more of a service listing.

Composition:

The listing is contained within a 3-column grid. In desktop view, the text description section on the left side takes up two columns, while an image, its caption, and a series of buttons takes up the other one-third. It is responsive, so for mobile devices, it will collapse into a single-column view, with text and images stacked vertically.

Launch Pad

Our launch pad on board the space station allows scientists to launch and dock small space probes, satellites, and manned capsules. Its state of the art technology design allows for multiple launches per day. Bookings may be made while booking the trip, or while at the space station.

Operational cost: $4600 per 24 hours

To use the launch pad, users must be affiliated with a professional research or engeneering organization or be in the posession of a space launch permit.

Specifications

  • Room size: 25m wide x 7.5 meters tall x 30 meters deep
  • Non-pressurized (space suit required)
  • 12 mechanical arms up to 15 meters long each
  • Observation deck for visitors to observe work within the pad
  • Unintended space drift protecting system
  • Unmanned rescue pods
  • Space launch support system
  • Space vehicle return guidance system
