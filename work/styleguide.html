Headings

CSS selectors

h1
h2
h3
h4

About headings:

There are 4 lairs of headings are used: h1, h2, h3, h4. H1 will lable different modules of the main site, H2 generally highlights the major sections of the site and major subsections of modules. H3 is used to highlight divisions in the subs sections such as what's a form what's a picture and so on. H4 is for other miscellaneous sections that would require bitter titles. For sections smaller than h4, use the em or strong tag to bold elements

Sample code for H1:


<div class="banner">
    <h1>Orbitar</h1>
</div>

Rendered Element:

Orbitar

Sample code for H2:


<h2>This is an H2 header</h2>

Rendered Element:

This is an H2 header

Sample code for H2 on dark backgrounds:


<h2 class="onDark">This is an H2 header on dark background</h2>

Rendered Element:

This is an H2 header on dark background

Information:

Sometimes the parent element will take care of coloring the child elements, such as this example. In case the parent element doesn't, the onDark class makes the element's font color white.
